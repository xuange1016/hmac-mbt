# hmac-mbt 项目简述

## 项目定位

`hmac-mbt` 是一个 MoonBit 实现的 HMAC 消息认证码库，首期聚焦 HMAC-SHA256。

项目目标是为 MoonBit 生态提供一个小而可靠的认证码基础组件，适用于接口签名、Webhook 校验、配置文件完整性校验和测试样例中的消息认证场景。

## 为什么值得做

MoonBit 生态仍在快速补齐基础库。HMAC 是很多协议和服务端集成的基础能力，算法标准清晰、测试向量充足，适合作为可复现、可验证的 MoonBit 基础组件。

## MVP 范围

- 提供 HMAC-SHA256 计算函数。
- 支持任意长度 key 和 message。
- 正确处理超过 SHA256 block size 的 key。
- 提供恒定时间 tag 比较函数。
- 使用公开测试向量验证核心行为。
- 提供 README 示例和基础 API 文档。

## 非目标

- 首期不实现 HMAC-SHA1、HMAC-SHA512 等多算法泛型接口。
- 首期不实现流式增量输入。
- 首期不承诺密码学安全随机数生成能力。
